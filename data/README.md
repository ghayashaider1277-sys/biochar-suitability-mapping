Data sources
This analysis uses the following publicly available datasets:
**SoilGrids 250m** — soil organic carbon, pH, bulk density  
  https://soilgrids.org
**SRTM DEM 30m** — elevation, slope, TWI  
  https://earthexplorer.usgs.gov
**Sentinel-2** — NDVI, land use/land cover  
  https://scihub.copernicus.eu
**Pakistan soil survey data** — field measurements
